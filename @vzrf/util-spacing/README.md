## Install

With [npm](https://npmjs.org/) installed, run

```
$ npm install @vzrf/util-spacing
```

## Usage

### HTML
```html
<link rel="stylesheet" href="node_modules/@vzrf/util-spacing/dist/index.css">
```

### SCSS
```scss
@import '@vzrf/util-spacing/src/index.scss';
```

# @vzrf/form

> VZRF Styles for form elements.

## Install

With [npm](https://npmjs.org/) installed, run

```
$ npm install @vzrf/form
```

## Usage

### HTML
```html
<link rel="stylesheet" href="node_modules/@vzrf/form/dist/index.css">
```

### SCSS
```scss
@import '@vzrf/form/src/index.scss';
```

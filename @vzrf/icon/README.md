# @vzrf/icons

> VZRF Icons

## Install

With [npm](https://npmjs.org/) installed, run

```
$ npm install @vzrf/icon
```

## Usage

### HTML
```html
<link rel="stylesheet" href="node_modules/@vzrf/icon/dist/index.css">
```

### SCSS
```scss
@import '@vzrf/icon/src/index.scss';
```
